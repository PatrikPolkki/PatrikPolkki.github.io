About me

I'm Patrik and I'm a Developer

I am Patrik, 21 years old student at Metropolia University of Applied Sciences. As a major, I study mobile development, but I can also code web pages. My main areas of expertise include Javascript, HTML, CSS, Node.js, Kotlin and React Native.

Creating new things made me excited about coding and learning a profession for myself from it. In coding, I am most fascinated by learning new things and getting better. Also, the possibilities of creation are limited to your own imagination, and that if nothing motivates.

I am also familiar with other coding languages, which we studied at school. In the first year, we coded Arduino Zumobot to follow a line and fight against other robots in C language. We also did a Mobile health application via Android Studio, with Java and in the game development we did a first-person fighting game, with Unity and C#.

In my second year studying mobile development as a major, I have made a word quiz app called Language Quiz in Kotlin language via Android studio. We also made social media website called Wildlife Collection, which includes a responsive design with HTML and CSS, and back-end REST API with Node.js. In the third project we made app called Officium, which is a job marketplace for odd jobs. Officium app was made in React Native.
